Notre site Web

Ce site web a été conçu dans le cadre du projet de Programmation Web à l’Efrei, avec pour objectif de promouvoir l’un des 17 objectifs de développement durable définis par l’ONU.

Dans le cadre de ce projet, nous nous sommes plus particulièrement concentrés sur l'ODD numéro 4. A savoir : Une éducation de qualité pour tous.

Notre projet

Dans le cadre de ce projet et de l'ODD choisit, nous avons donc décidé de confectionner un site regroupant toutes les annales des tests de l'année afin de rester dans la démarche d'une éducation de qualité et plus facile d'accès pour tous les étudiants.
